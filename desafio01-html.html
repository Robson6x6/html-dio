HTML

(inicio)

Ao contrário do que muitos pensam HTML não é uma linguagem de programação e sim de marcação.
Seu significado Hyper Text Markup Languadge e é a linguagem base para criação de páginas web.
Utiliza as "tags para definir os seus conteúdo."

Tags

(inicio)

As tags são comandos que ficam dentro do sinal de maior e menor (< >) que definem o do que se trata o texto.
temos a tag <p> porém sem os espaços que definem a tag de paragrafo.
existem tags que tem abertura e fechamento (<h1> ... <h1> tags de títulos por exemplo) e existem tags que nao precisam de fechamento (<br> utilizada para pular uma linha.)

Títulos

(inicio)

Existem 6 niveis de título que vão do H1 ou H6.
Uma comparação seria que: o <h1> é o mais importante e o <h6> o menos importante.
São essencias para organizar e ajuda na otimização para os buscadores.

Listas

(inicio)

Existem dois tipos de lista que são:

Lista ordenadas

  1. Exemplo
  2. de
  3. lista
  4. ordenada

lista não ordenada

  • Exemplo
  • de
  • lista
  • não
  • ordenada

Links

Os links são criados pela tag <a> e permitem conectar as páginas ou abrir arquivos
ela possui o atribito herf que define o seu destino e também o atributo target="_blank" que abre o link em uma nova guia

Algumas Outras tags

(inicio)

<del>

usada para indicar que um trecho foi removido
aparece Riscado

<abbr>

Usado para explicar um sigla ou abreviação juntamente com o atributo title
pouse o mouse em cima da sigla >>> HTML <<<

perceba que foi criado uma caixa de texto com seu significado.

<blockquote>

indica uma sitação podendo ser de um autor ou font geralmente possui um recuo automatico quando utilizado pode ser acompanhado do atributo cite que permite informar a fonta da citação
exemplo:

"Ô promessa desgraçada, ô promessa sem jeito!"
